## [0.1.0] - 2026-06-28

### Added
*   Repository scaffolding (`.specify/`, `docs/`, `agent/`, `src/`, `data/samples/`).
*   Spec Kit initialization with official GitHub Spec Kit (`specify-cli`).
*   Specs created: `spec.md` (What & Why), `plan.md` (checkpoints/milestones), `data-model.md` (nested schemas), and `tasks.md` (backlog assignments).
*   Non-negotiable constitution enforcing 100% offline, CPU-only, AGPLv3 compliance.
*   Two-person team division of labor (`WORK_DIVISION.md`) with sync-point checkpoints between Sharique and Harshit.
*   Backlog of 8 development issues documented in `docs/ISSUES.md`.
*   GNU AGPLv3 `LICENSE` file.
*   FastAPI backend entrypoint (`src/main.py`) and smoke testing module (`src/test_smoke.py`).
*   CI/CD configuration (`.gitlab-ci.yml`) including lint, formatting, type check, security scans, unit tests, and commit message quality gates.
*   Contributor guidelines (`CONTRIBUTING.md`).
