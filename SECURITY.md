# Security Policy

## Supported Versions

We actively support and patch the following versions of MedScribe:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you find a security vulnerability or secret exposure, please do not file a public issue. Instead, report it responsibly:

1. Send an email detailing the vulnerability to `security@medscribe.local` or the project maintainer.
2. Provide a description of the issue, step-by-step instructions to reproduce it, and any potential remediation.
3. We will acknowledge receipt of your report within 48 hours and work on a patch within 14 days.
